# git # clone # 개발자 # Fork # 신입개발자 # remote # gitLab

#gitLab 사진

# clone 하는 방법(2가지)

1. git bash에서 작성

  1. 작업 폴더로 이동하여 오른쪽 마우스를 클릭하여 ‘git bash here’로 들어간다 또 git bash 접속해서 cd를 통하여 디렉터리를 찾아서 접속한다. cd GitTest mkdir 폴더명 디렉터리 하나를 만든다.
  2. 공용 GitLab 저장소에서 clone 버튼 클릭해서 SSH주소를 복사한다.
  3. Clone
git clone [SSH주소]

2. Goland에서 작업

  1. open클릭-Users/bist/새폴더 하나 만들어서 선택-OK클릭
  2. clone 명령어 쓰기
git clone [SSH주소]
  1. 폴더명 작성 후 폴더 안으로 이동하기
cd sample-git

# 원격 저장소 설정(GoLand에 작업!)

  1. 원격 저장소 별칭 내역을 보여주는 명령어
git remote -v
  1. 저장소 이름 변경
  • origin은 upstream으로 이름을 변경합니다.
git remote rename origin upstream
  1. Fork 하기
  • fork한 원격 저장소를 local과 연결해서 데이터를 복사하는 작업
방법: git remote add [별칭] [fork된 주소]
예시: git remote add genie git@gitlab.gantry.ai:genie/sample-git.git
