🐾   [masOS] 맥북에 aws cli 설치하기

2022/09/26

# Mac # MacOS # aws # aws cil # MacOS # 맥북 # 설치

배경

aws 정보를 로컬에 셋팅하려고 터미널을 열었다. 그런데 아래와 같은 메시지가 떴다.

command not found: aws

설치를 해야하는 구나… 윈도우에서 git bash에서 했지..

설치 방법

  1. 터미널에 아래와 같은 명령을 친다
curl "https://awscli.amazonaws.com/AWSCLIV2.pkg" -o "AWSCLIV2.pkg"
  1. 아래 명령어를 친다
sudo installer -pkg AWSCLIV2.pkg -target /

  1. 비번을 치라고 하면 컴퓨터 비번을 치면 된다

  2. 설치 성공 메시지가 나온다

    installer: Package name is AWS Command Line Interface installer: Installing at base path / installer: The install was successful.

  3. 설치 위치 확인하기

which aws  // /usr/local/bin/aws
  1. 버전 확인하기
aws --version

참조

https://docs.aws.amazon.com/ko_kr/cli/latest/userguide/getting-started-install.html
